Eigenschappen block boxes:

  • Een block element pakt de hele horizontale ruimte die beschikbaar is, ongeacht hoeveel het nodig heeft. Informatie die na het block element komt, wordt automatisch naar een nieuwe regel geduwd. Het is wel te voorkomen door speciale maatregelen zoals floating.
  • Voorbeelden van block tags zijn, < div> (standaard block element), header tags < h1> t/m < h6>, paragraphs < p>, unorderd list < ul> en tabellen < table>.

Eigenschappen inline boxes:

  • Een inline element pakt precies genoeg horizontale ruimte om zijn inhoud weer te geven.
  • Voorbeelden van inline tags zijn < span> (standaard inline element), hyperlinks (anchors) < a href=..>, afbeeldingen < img> en invoerlementen < input>.

    Opdracht 4

    • Content: content is de inhoud.
    • Padding: padding is de opvulling aan de "binnenkant" van de 4 randen van je element
    • Margin: margin zijn de blanco ruimtes gemeten van de vier randen van de pagina tot aan de inhoud. Met andere woorden een ontzichtbare grenslaag rond je element.
    • Border: borders zijn de randen rondom je inhoud. Dit kunnen dikke of dunne randen zijn, stippelijn etc. De borders kunnen ook een kleur krijgen.

    De CSS property wat je kan gebruiken om ervoor te zorgen dat de waarde van widht en height de totale breedte en hoogte is van het blok is de box-sizing property:border-box. Om dit meteen voor je hele pagina te laten gelden gebruik je * { box-sizing:border-box} bovenaan je css file.
