Opdracht 1

  • Static: Elementen zijn standaard static. Ze zijn niet affected bij top, bottom, right, left properties. Static volgt de pagina
  • Relative: Relative gedraagt zich hetzelfde als static tenzij je nog een aantal extra properties opneemt.
  • Fixed: Fixed gedraagt zich dat het element op dezelfde plek blijft staan, ongeacht of er door de pagina wordt gescrold
  • Absolute: Absolute gedraagt zich als fixed, maar wordt gepositioneerd bij de dichtsbijzijnde ancestor element. Als er geen ancestor is, dan wordt de body gebruikt en zal het meescrollen met de pagina.
  • Sticky: Sticky is gepositioneerd op basis van de gebruikers scroll positie. Voor sticky moet je top, right, bottom of left gebruiken.

    • Dit element is static
    Dit element is relative;
    Dit element is absolute
    Dit element is fixed
    Dit element is sticky
