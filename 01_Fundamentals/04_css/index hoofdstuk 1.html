Opdracht 1

3 manieren om CSS in te laden aan je HTML pagina

  • External
  • Internal
  • Inline

    • External: Een externe stylesheet koppelen aan een HTML pagina. Je gebruikt de tag < link> in HTML na je titel, binnen de head.

    Internal: Een interne stylesheet maak je binnen je HTML pagina. Je gebruikt de tag < style> binne de head sectie van je HTML pagina.

    Inline: Een inline style gebruik je als je een unieke stijl wilt toevoegen aan 1 element. Je gebruikt de tag < style> dan binnen het element wat je wil veranderen binnen je HTML pagina

    Opdracht 2

    Een CSS syntax bestaat uit een selector, property en een value een voorbeeld: p { color: red; text-align: center; }
    de p verwijst naar het stukje met de p tag in HTML ( de selector ). { openingtag en de } is de closingtag. Color is je property en de kleur in je value. Hetzelfde geldt voor text- align, dat is je property en je value is center

    Opdracht 3

    Type selectors:

  • * is de universele selector. Dit geldt voor de hele HTML pagina
  • Je hebt ook een specifieke selector zoals de p element of het div element.
  • De id selector staat voor het selecteren van een element met een specifieke id je gebruikt dan de # ervoor.
  • Het class element gebruik je om het element met een specifieke class te selecteren. je gebruikt dan een . gevolgd door de classname
  • Je kan ook een groep selecteren. Dan gebruik je een komma tussen de elementen
  • De descendant selector komt overeen met alle elementen binnen 1 element. Bijvoorbeeld div p. Hier gaat het om het selecteren van alle p elementen binnen de div elementen

    • Voorbeelden selectors:

  • id selector: #firstname { background-color: yellow; }
    Style het element met de id firstname
  • class selector: .intro { background-color: yellow; } de class is intro, . betekent alle class met intro selecteren
  • p.hometown { background-color: yellow; } alle p elementen met de class hometown stylen
  • < p class=" center large ">This paragraph refers to two classes.< /p> het element is p en heeft twee elementen, namelijk center en large
  • id selector: #firstname { background-color: yellow; }
  • Descendant selector: Je hebt bijvoorbeeld een div tag ( met id main ) en daar binnenin verschillende p tag en misschien ook een div tag. Dan zijn de p en div descendants van de ( eerste ) div ( met id main ). Een voorbeeld: main p { background-color: yellow; }
